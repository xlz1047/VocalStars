# VocalStars Project Overview

## Vision
A singing evaluation platform that provides fair, musical, and meaningful feedback. Unlike traditional karaoke apps that penalize expressive variation, VocalStars focuses on how a person sings, allowing for natural pitch movement and style.

## Core Features
1. **Vocal Recording**: Mobile/web interface for singing along to a reference melody.
2. **Pitch Visualization**: Supportive visual representation of singer's pitch vs. reference.
3. **Performance Evaluation**: Multi-dimensional assessment (accuracy, timing, stability, expressiveness).
4. **Feedback Summary**: Overall score, category breakdown, and written explanations.
5. **Playback Review**: Replay recording with visual feedback overlays.

## Design Goals
- **Supportive & Encouraging**: The UI should feel like a coach, not a judge.
- **Musical & Expressive**: Use fluid visualizations rather than rigid bars.
- **Clear & Actionable**: Feedback should help the user understand their strengths and growth areas.
